# 环境变量各项参数说明
 - `DATABASE_URL` 数据库
 - `HOST` 网站域名
 - `LIMIT_TIMES` 在请求间隔里所能请求的次数
 - `LIMIT_TIME` 请求间隔(秒)
 - `LIMIT_WAIT_TIME` 超出等待时间
 - `MAX_URL` 最大URL长度
 - `MAX_SHORTCODE` 自定义短代码最大长度
 - `MIN_SHORTCODE` 自定义短代码最小长度
 - `RAND_SHORTCODE` 随机短代码长度
 - `WEB_BACKGROUND` 网页背景图片
 - `ADMIN_URL` 后台链接
 - `ADMIN_LOGIN_PASS` 后台登录密码
